Moving Car Animation

A visually engaging web animation that simulates a moving car using CSS animations and JavaScript, demonstrating smooth motion effects and interactive UI behavior.

Features
- Smooth car movement animation using CSS keyframes
- Animated background (road/sky effect) for realistic motion
- Lightweight and responsive design
- Continuous looping animation for seamless experience
  
Tech Stack
HTML5
CSS3 (Animations & Keyframes)
JavaScript (optional enhancements)

How It Works
- CSS @keyframes is used to animate the car and background elements
- Background layers move at different speeds to create a parallax effect
- JavaScript (if used) controls animation triggers or interactivity
